## Variant: Stripe Editorial — Premium White

### Design stance
Stripe-inspired elegance with weight-300 typography, airy whitespace, and a single deep-teal chromatic pulse. The palette is mostly achromatic (white, navy, gray) with one intentional color moment.

### Key choices
- **Typography**: Serif headlines (Source Serif 4 400) + lightweight sans body (Inter 300) — editorial, confident, calm
- **Color**: White canvas, deep navy (#061d29) headings, slate (#64748b) body text, deep-teal (#006e75) as the only accent
- **Layout**: Generous whitespace (80-120px vertical sections), centered hero, 3-column feature grid
- **Shadows**: Blue-tinted (rgba(50,50,93,0.25)) — Stripe-style elevation
- **Radius**: Conservative 6-8px, subtle and professional
- **Badges**: Mint glass kept but softened for GEMS badge only

### Trade-offs
- **Strong at**: Professional, trustworthy, editorial — feels like a premium publication
- **Weak at**: Not playful — may feel too serious for some younger audiences

### Best for
Teachers who want a trustworthy, high-end resource that feels authoritative and premium
